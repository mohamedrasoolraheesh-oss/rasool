Stock Price Prediction Using Linear Regression

A machine learning project that predicts stock closing prices using historical stock market data and the Linear Regression algorithm from Scikit-learn.

Features

* Load and preprocess stock market dataset
* Train a Linear Regression model
* Predict stock closing prices
* Evaluate model performance using:
    * Mean Squared Error (MSE)
    * R² Score
* Visualize actual vs predicted prices using Matplotlib

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
